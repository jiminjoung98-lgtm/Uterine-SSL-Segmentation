**"Semi-Supervised Segmentation for Precise Uterus and Endometrium Analysis in Ultrasound Imaging"**

This repository provides:
- ssl4Ultrasound: A semi-supervised segmentation framework for uterine ultrasound imaging
- seg4Ultrasound: Multi-head segmentation for uterus, endometrium and nicheanatomical structures
